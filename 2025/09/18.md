## わかったこと
- 入れ子の辞書は`list(dict.values())`で辞書が要素のリストの形に展開できること。
```
{
  "2": {"id": "テスト商品2", "name": "テスト商品2"},
  "7": {"id": "テスト商品7", "name": "テスト商品7"},
}
```
は
```
[
    {"id": 2, "name": "テスト商品2"},
    {"id": 7, "name": "テスト商品7",
]
```
となる。
- htmlでは演算は行わないこと。views.pyで演算した結果をcontextにつめて渡す。
